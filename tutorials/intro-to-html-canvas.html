Intro to HTML Canvas

HTML Canvas is a very powerful tool for all web developers. It's also really easy to learn, and unlocks infinite possibilities - you can make ultra-animated elements, interactive backgrounds, or even full-fledged games!

The Basics

HTML Canvas is a DOM element, which means it can be included in HTML files manually. It can also be created dynamically, which we'll explore later.

We've now created a canvas element! The width and height just tells the browser how big the canvas should be.

Then, we create a script.js file and reference the canvas.

const canvas = document.getElementById("canvas"); const ctx = canvas.getContext("2d"); canvas.width = 100; canvas.height = 100;

The way HTML Canvas works is through its context. We called our context ctx in this instance, but you can call it whatever you want. You tell its context what to draw, where to draw, etc.

We've also defined our canvas' width and height. If we don't do that, our canvas could be 0x0, or something else if the browser chooses.

Now let's try drawing a rectangle:

ctx.fillStyle = "white" ctx.fillRect(0, 0, canvas.width, canvas.height); ctx.fillStyle = "red" ctx.fillRect(0, 0, 80, 50);;

The first value is the x-coordinate of the top-left corner.
